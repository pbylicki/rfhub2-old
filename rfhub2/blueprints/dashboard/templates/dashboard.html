Robot Framework Hub

You are viewing a very (very!) early preview of the robot hub -- a web site and web service that provides web-based access to your test assets.

In its present form, the hub has two primary services:

  • A web application for viewing keyword documentation (view documentation for built-in, installed, and local keyword libraries and resource files).
  • A web api for fetching keyword information. This can be used by other tools that need access to your test assests. For example, a text editor can use information from this API in order to provide auto-completion.

Note: if you don't want to see this page when you go to the rfhub root (eg: "http://localhost:7070/"), you can start rfhub with the "--root /doc" option to have it always start on the documentation page. 

$ rfhub --root /doc
